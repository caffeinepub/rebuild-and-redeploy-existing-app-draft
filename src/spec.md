# Member Rewards Platform

## Overview
A membership platform where users authenticate with Internet Identity to access features, earn points through participation, and trade points for products in a marketplace.

## Authentication
- Users must log in using Internet Identity before accessing any application features
- All functionality is restricted to authenticated users only

## Membership System
- Users can become members after authentication
- New members automatically receive welcome tokens/points upon joining
- System tracks member participation for future point-awarding activities

## Member Profiles
- Each member has a profile page displaying their information
- Members can upload and display a custom avatar image
- Profile shows member status and current point balance

## Points System
- Members earn points for joining the platform
- System designed to support future expansion for awarding points based on activities like:
  - Reading articles
  - Attending events
  - Other participation activities
- Points balance is tracked per member

## Marketplace
- Dedicated marketplace page where members can browse available products
- Members can trade their earned points for various products
- Product catalog with point costs displayed
- Transaction system to exchange points for products

## Backend Data Storage
- Member profiles and authentication data
- Point balances and transaction history
- Product catalog with point costs
- Member avatars and profile information
- Participation tracking data for future point-awarding features

## Core Operations
- Member registration and profile creation
- Point allocation and balance management
- Product catalog management
- Point-to-product exchange transactions
- Avatar upload and storage
- Member data retrieval for profiles and marketplace access
